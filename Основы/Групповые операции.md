**Выбор уникальных элементов столбца**

SELECT DISTINCT <название_стб> FROM <таблица>; //вывод уникальных элементов
SELECT <название_стб> FROM <таблица> GROUP BY <название_стб>; //группировка данных по значению

**Групповые функции SUM и COUNT**

SUM(<название_стб>) GROUP BY <название_стб> //суммирование
COUNT(<название_стб>) GROUP BY <название_стб> //сколько записей относится к группе
COUNT(*) —  подсчитывает  все записи, относящиеся к группе, в том числе и со значением NULL;
COUNT(имя_столбца) - без NULL

**Групповые функции MIN, MAX и AVG**:
- MIN(<название_стб>) - минимальное значение столбцов, относящихся к группе
- MAX(<название_стб>) - максимальное значение столбцов, относящихся к группе
- AVG(<название_стб>) - среднее значение столбцов, относящихся к группе

**Выборка данных c вычислением, групповые функции**:
- В качестве аргументов функции можно использовать арифметические выражения

**Вычисления по таблице целиком**:
- С помощью групповых функций можно вычислять значения по всей таблице (без GROUP BY)

**Выборка данных по условию, групповые функции**:
- Ключевое слово HAVING после GROUP BY
- Без GROUP BY - WHERE

**Порядок выполнения запроса**:
- FROM
- WHERE
- GROUP BY
- HAVING
- SELECT
- ORDER BY
