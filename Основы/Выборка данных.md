**Отбор всех данных**

SELECT * FROM <название_таблицы>;

**Выборка отдельных столбцов**

SELECT <стб1, стб2> FROM <название_таблицы>;

**Присвоение новых имен столбцам при формировании выборки**

SELECT <название_стб> AS <новое_название_стб>, ... FROM <название_таблицы>;

**Выборка данных с созданием вычисляемого столбца** 

Подразумевается вычисление по строкам. Можно использовать арифметические операции.

**Выборка данных, вычисляемые столбцы, математические функции**

Функции:
- CEILING(x) - округление в большую сторону;
- ROUND(x, k) - округление x до k знаков после запятой;
- FLOOR(x) - округление в меньшую сторону;
- POWER(x, y) - возведение в степень;
- SQRT(x);
- DEGREES(x) - перевод из градусов в радианы;
- RADIANS(x) - наоборот;
- ABS(x) - модуль;
- PI().

**Выборка данных, вычисляемые столбцы, логические функции**

Функции:
- IF(логическое_выражение, выражение_1, выражение_2)

**Выборка данных по условию**

SELECT <стб1, стб2> FROM <название_таблицы> WHERE <условие>;
WHERE может включать :
- операторы сравнения (=, >, <, >=, <=, <>);
- операции математические;
- логические операции (И «and», ИЛИ «or», НЕ «not»).

**Выборка данных, операторы BETWEEN, IN**:
- BETWEEN <число> AND <число>
- IN (<список_значений>)

**Выборка данных с сортировкой**

ORDER BY <название_стб>/<номер_столбца> (DESC - по убыванию)
По умолчанию, сортировка идет по возрастанию.

**Выборка данных, оператор LIKE**

Используется для сравнения строк на совпадение с шаблоном:
- SELECT * FROM <название_таблицы> WHERE <название_стб> LIKE '%М.%' - любая строка;
- SELECT * FROM <название_таблицы> WHERE <название_стб> LIKE 'Поэм_' - любой одиночный символ.
